# Assessment 3: Azure Static Website IaC Deployment

This project demonstrates the deployment of a simple static web application on Microsoft Azure using Terraform.

## Project Overview

The infrastructure includes:
- Azure Resource Group
- Azure Storage Account
- Azure Static Website hosting
- Uploaded index.html file

Terraform was used to define and deploy the cloud infrastructure as code.

## Tools Used

- Microsoft Azure
- Azure Cloud Shell
- Terraform
- Azure Storage Static Website

## Deployment Output

Resource Group: rg-assessment3-iac-webapp  
Storage Account: iacwebc9ti3i  
Static Website URL: https://iacwebc9ti3i.z8.web.core.windows.net/

## Repository Structure

- app/: contains the static website file
- terraform/: contains Terraform configuration files
